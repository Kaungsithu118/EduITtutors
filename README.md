## Project Overview
EduITtutors is an online IT tutoring platform designed to provide structured and accessible learning resources for students. The website offers IT courses, webinars, blogs, and administrative tools to support both learners and instructors.

## Key Features
- Online IT courses with structured lessons
- Webinar and event management
- Blog and educational content publishing
- User authentication and profile management
- Admin dashboard for managing courses and content

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Tools: GitHub, GitHub Desktop

## System Modules
- User Module (registration, login, profile)
- Course Management Module
- Webinar & Event Module
- Blog Management Module
- Admin Panel

## Project Purpose
This project was developed for academic and learning purposes to enhance practical skills in web development, database management, and system design.

## Installation & Setup (Optional)
1. Clone the repository
2. Configure database connection
3. Import the provided SQL file
4. Run the project on a local server (XAMPP / WAMP)
